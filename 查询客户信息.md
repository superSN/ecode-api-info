# 查询客户信息

## 调用说明：
```
接用方式：GET
接口地址：
http://api.1code.cc/Custom/info?access_token=ACCESS_TOKEN&appid=APPID&cid=CID
```

## 参数说明：
| 参数| 是否必须 | 说明 |
| -- | -- | -- |
| appid | 必须 | 第三方用户唯一凭证 |
| access_token | 必须 | 由全局接口获得 |
| cid | 必须 | 客户编号 |


## 返回说明
参数说明：

| 参数 | 说明 |
| -- | -- |
| access_token | 获取到的access_token凭证 |
| expires_in | 凭证有效时间，单位：秒 |
